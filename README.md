# Brand, not slop

Workshop deck — **Laura Rosa × Ivan, 2026**. How to use AI to *execute* a brand
with clear, coherent rules instead of generating slop.

**View it live:** https://tomacco.github.io/brand-not-slop/

Or clone and open `index.html` — the deck is fully self-contained (no network
needed, no build step).

## Controls

| Key | Action |
| --- | --- |
| `→` / `←` | Next / previous (some slides have presenter beats on `→`) |
| `F` | Fullscreen |

Append `?print=1` to the URL for a flat, animation-free version suitable for
printing to PDF.

## Contents

- `index.html` — the 34-slide deck, including the full prompt chain in the appendix
- `samples/` — standalone type-direction specimens shown during the workshop
