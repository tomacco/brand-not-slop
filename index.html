Appendix · take-home

The seven prompts

Act as a brand strategist. I will describe an idea. Build its brand DNA.

My idea: [DESCRIBE YOUR IDEA HERE]

Return exactly these sections:
1. ONE SENTENCE — if this brand had to explain itself in one sentence, what would it say? Every later decision answers to this line.
2. PURPOSE / AUDIENCE / PROBLEM / PROMISE — one short paragraph each.
3. VALUES — 3 to 5, each with the behavior that proves it.
4. PERSONALITY — 3 to 5 adjectives, each defined in one line. An undefined adjective drifts.
5. TONE — how the brand speaks, with one example sentence.
6. DESIRED PERCEPTIONS — what a stranger should think after five seconds.
7. GUARDRAILS — allowed visual signals, banned visual signals, allowed words, banned words. Write every rule as a rule about meaning ("every color must read as warm"), never as a count ("max 3 colors"). Give the reason for every ban.

If my idea is missing information, state your assumption explicitly instead of asking me.
Here is a brand DNA. Translate its personality into perceivable signals.

Brand DNA: [PASTE THE FULL OUTPUT OF PROMPT 1]

For each personality adjective, return a row with four columns:
- ATTRIBUTE — the adjective.
- VISUAL BEHAVIOR — the shapes, spacing, weight, color temperature, and density it implies. Example: "warm" = open spacing, soft curves, warm hues, generous line height.
- VERBAL BEHAVIOR — the sentence length, vocabulary, capitalization, and rhythm it implies.
- BETRAYAL — the single most likely signal that would contradict this attribute in practice.

Stay inside the guardrails from the DNA. If an attribute and a guardrail conflict, flag the conflict explicitly. Do not silently resolve it.
Act as a brand strategist. Using this brand DNA and signal map, propose 3 positioning territories.

Materials: [PASTE THE OUTPUTS OF PROMPTS 1 AND 2]

For each territory:
- NAME — two or three words.
- THE CROWDED DEFAULT — the generic position this territory deliberately avoids.
- WHO IT ATTRACTS — and who it repels. A position that repels nobody positions nothing.
- PERCEPTION IT CREATES — one sentence.
- VISUAL LANGUAGE — which signals from the map it leans on hardest.
- INCOHERENCE RISK — where this territory is most likely to break the guardrails.

Do not recommend one. I choose.
Build the brand voice and messaging system for the territory I chose.

Chosen territory: [PASTE IT]
Guardrails, including banned words: [PASTE THEM FROM PROMPT 1]

Return:
- TAGLINE — 3 options, all inside the guardrails.
- BIO — under 280 characters, ready for a social profile.
- HOMEPAGE HEADLINE + SUBLINE.
- CTA STYLE — how buttons and asks sound, with 3 examples.
- 5 SENTENCES THE BRAND WOULD SAY — in real situations: announcing, apologizing, selling.
- 5 SENTENCES IT WOULD NEVER SAY — each with the specific rule it breaks.

Treat the banned-words list as literal: none of those words may appear anywhere. Use plain language. If a 12-year-old would not understand a line, rewrite it.
Generate 3 visual directions for this brand.

Materials: [PASTE THE DNA, THE SIGNAL MAP, AND THE CHOSEN TERRITORY]

For each direction, specify: palette (with color temperature), typography (described as a voice, never just font names), contrast level, shapes, density, composition, and photography or texture treatment.

Rules:
- Every choice must trace back to a personality attribute from the signal map. Name the attribute next to the choice.
- Exclude anything that contradicts the primary attribute, even if it looks good.
- For each direction, add MISREAD RISK: what a stranger might wrongly conclude in five seconds, and which signal would cause it.

I will choose one direction. Do not blend them for me.
Condense this brand into a one-page brand system I can hand to any designer, writer, or AI.

Materials: [PASTE THE OUTPUTS OF PROMPTS 1–5, INCLUDING THE CHOSEN DIRECTION]

Sections: brand principles (the one sentence first), voice summary, palette, typography, spacing, iconography, photography, motion, and usage rules.

Every rule must be a meaning rule that survives a redesign ("every color reads as warm"), never a count rule ("max 3 colors"). For each signal, add one DO and one DON'T, each with its reason.

End with THE TEST: print the one sentence, followed by this instruction: "Any future asset that cannot answer to this sentence gets rejected."
Act as a hostile brand auditor. Your job is to find what breaks. Reassurance is a failure state.

Brand system: [PASTE THE OUTPUT OF PROMPT 6]

1. Check positioning, voice, and visual direction against each other. List every contradiction, ranked by severity, each with one concrete fix.
2. Check every rule against the primary personality attribute. Flag any signal that betrays it — reuse the BETRAYAL column from the signal map if I provided it.
3. Run the survival checklist: does the system still work at 16 pixels, in grayscale, in dark mode, on a cracked phone screen in sunlight, and in a screenshot with no caption? Flag what fails and why.

When I paste a revised system, audit it again from zero. Stop only when a full pass finds nothing new.
